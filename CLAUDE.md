# AGDS — Project Instructions

## Language Policy

All documentation, source code, and code comments in this repository **must be
written in English**. This includes:

- Markdown documents under `docs/` and any README files
- Source code identifiers (variables, functions, types, files)
- Inline and block comments
- Commit messages and pull request descriptions
- CLI help text, log messages, and error messages

Conversations with the user may be conducted in the user's preferred language,
but any artifact committed to the repository must be in English.
